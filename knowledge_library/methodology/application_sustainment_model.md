# Application sustainment and support delivery model

Fujitsu delivers application sustainment services through a structured, ITIL-aligned framework focused on stability, performance and continuous improvement.

The model includes:
- Level 2 and Level 3 application support
- Change and release management
- Problem management and root cause analysis
- Stakeholder and supplier coordination

Fujitsu applies:
- Proactive monitoring and maintenance
- Preventative problem management
- SLA-driven performance tracking

The approach ensures:
- Stable and reliable application performance
- Continuous service improvement
- Alignment with Defence operational requirements
