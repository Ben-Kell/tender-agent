# Embedded security model for Defence ICT solution delivery

Security is embedded across all phases of Fujitsu’s delivery approach to ensure protection of sensitive Defence information and compliance with government and industry standards.

Security controls include:
- Role-based access control
- Data encryption mechanisms
- User authentication and authorisation controls

Security considerations are incorporated into:
- Solution design
- System integration
- Operational delivery

These controls ensure that Defence data is protected across all environments while maintaining compliance with regulatory and policy requirements.

Security is treated as a foundational component of delivery rather than an overlay, ensuring consistent protection across the full lifecycle of ICT services.
